# **App Name**: StudyAI

## Core Features:

- User Authentication: Secure user registration and login with JWT and password hashing.
- Notes Management: Create, retrieve, and delete notes.  All notes are stored in MongoDB.
- Flashcard Generation: Automatically generate flashcards from user notes using an AI tool.
- Flashcard Management: Save flashcards and mark them as mastered to track progress.
- Study Planner: Create study goals and set deadlines.
- Progress Tracking: Automatic streak tracking and goal completion.
- AI Chat Assistant: Chat with an AI assistant for study help, saving chat history linked to the user.

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB) to evoke a sense of calm and focus.
- Background color: Light gray (#F0F0F0), offering a clean and distraction-free interface.
- Accent color: Soft Lavender (#E6E6FA) to highlight important elements and CTAs.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral look.
- Use simple, outlined icons for intuitive navigation.
- Clean and structured layout with clear sections for notes, flashcards, and study plans.
- Subtle animations and transitions to provide feedback and enhance the user experience.
